# methode: `list.append()`
Als je een lijst wil uitbreiden met een nieuw element, gebruik je de methode `append`. Je geeft het element dat je wil toevoegen mee als argument.

Bijvoorbeeld:
```python
a = []
a.append(1)
a.append(2)
a.append(3)
# nu is het alsof we a = [1,2,3] hadden geschreven
```
